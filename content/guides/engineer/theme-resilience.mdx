# レジリエンス・BCPに使える経営工学キーワード

## 1. このテーマが技術士試験で重要になる理由

近年は、災害、感染症、地政学リスク、サイバー攻撃、物流制約など、企業活動を止める外乱が増えています。経営工学部門では、効率性だけでなく、止まりにくさ、復旧力、代替手段を設計する力が問われます。

## 2. 社会背景・白書背景

サプライチェーン強靭化、経済安全保障、事業継続、サイバーセキュリティ、人手不足、物流制約が背景になります。白書背景としては、リスク分散、デジタル化、BCP策定、安定供給と接続できます。

## 3. このテーマで使える主要キーワード

| キーワード | 答案での使いどころ |
|---|---|
| [レジリエンス](/guides/engineer/resilience) | 外乱に対する耐性と復旧力を示す |
| [BCP](/guides/engineer/bcp) | 重要業務、RTO、代替手段、訓練を整理する |
| [SCM](/guides/engineer/scm) | 供給網全体のリスクを管理する |
| [需給調整](/guides/engineer/demand-supply-adjustment) | 需要変動と供給制約を調整する |
| 在庫最適化 | 欠品リスクと在庫コストを調整する |
| モーダルシフト | 輸送手段の代替性を高める |
| データドリブン | リスク兆候や在庫状況を見える化する |
| サイバーリスク | システム停止や情報漏えいへの備えを示す |
| EDI | 取引先との情報連携を安定化する |
| TMS | 輸送状況と代替ルートを管理する |

## 4. 答案での使いどころ

背景では、外乱が発生したときに供給停止や納期遅延が起こる状況を示します。課題では、重要業務の未定義、代替手段不足、取引先依存、復旧訓練不足を挙げます。

## 5. 課題抽出に使える視点

- 重要業務と復旧優先順位が明確でない
- 単一サプライヤ、単一物流ルートに依存している
- 在庫、需要、輸送状況が見えない
- BCPが文書だけで訓練されていない
- サイバー攻撃時の代替運用がない

## 6. 解決策に使える視点

- BIAを行い、重要業務とRTOを設定する
- 代替サプライヤ、代替輸送、在庫バッファを設計する
- EDI、TMS、WMSにより状況を見える化する
- 訓練とレビューによりBCMとして運用する

## 7. リスクと対策に使える視点

冗長性を高めると、在庫費用や管理負荷が増えます。重要度に応じて対象を絞り、KPIで効果を確認し、定期的にリスクシナリオを見直す必要があります。

## 8. 技術者倫理・社会の持続可能性への接続

レジリエンスは、顧客、地域、社会への安定供給に関わります。技術者は、短期効率だけでなく、公益、従業員安全、取引先との公平性、事業継続を踏まえて設計します。
