# 技術士 経営工学 問題演習

## 1. このページの役割

このページは、技術士 経営工学部門の問題演習入口です。過去問トレンド、設問形式、重要キーワードを確認したあと、実際に答案骨子を作り、答案練習へ進むために使います。

トレンドマップは「何を学ぶか」を決めるページ、このページは「選んだテーマでどう練習するか」を決めるページです。

## 2. このページでできること

- テーマから演習テーマを選ぶ
- 設問形式から答案の型を選ぶ
- 使うキーワードを選ぶ
- 答案骨子の作り方へ進む
- 600字・1600字答案練習へ進む準備をする

## 3. このページでやらないこと

- 公式過去問本文の全文表示
- 自動採点
- 本格的な答案添削
- 動的な問題生成

これらは将来的な拡張対象とします。まずは、テーマ、設問形式、キーワードを選び、答案骨子へ進む入口として使います。

## 4. 演習の進め方

1. テーマを選ぶ
2. 設問形式を選ぶ
3. 使うキーワードを3〜5個選ぶ
4. 答案骨子を作る
5. 600字または1600字答案へ展開する
6. リスク、倫理、持続可能性の観点を確認する

## 6. 設問形式から演習する

| 設問形式 | 演習でやること | 参照ページ |
|---|---|---|
| 多面的な課題を抽出する | 課題を3つに分ける | [設問形式別の問われ方](/guides/engineer/past-exam-question-patterns) |
| 最重要課題を選ぶ | 理由を明確にする | [答案骨子の作り方](/guides/engineer/answer-structure-guide) |
| 解決策を示す | キーワードを施策化する | [答案用途別キーワードマップ](/guides/engineer/keyword-answer-uses) |
| リスクと対策を書く | 副作用と対応策を整理する | [答案骨子の作り方](/guides/engineer/answer-structure-guide) |
| 倫理・持続可能性を書く | 公衆の安全、説明責任、環境・社会への影響を整理する | [白書背景×キーワード対応表](/guides/engineer/whitepaper-keyword-map) |

## 7. キーワードから演習する

キーワードから演習する場合は、まず個別キーワードページで定義・課題・対策を確認し、次に答案用途別キーワードマップで答案のどこに配置するかを確認します。

- [キーワードマップ](/guides/engineer/keywords)
- [答案用途別キーワードマップ](/guides/engineer/keyword-answer-uses)
- [最重要キーワード100](/guides/engineer/keyword-priority-100)

## 8. 答案骨子へ進む

テーマ、設問形式、キーワードを選んだら、答案骨子の作り方へ進みます。答案骨子では、背景、課題、最重要課題、解決策、リスクと対策、倫理・持続可能性の6ブロックで整理します。

- [答案骨子の作り方](/guides/engineer/answer-structure-guide)
