# 答案練習を始める

このページでは、技術士二次試験の答案練習を始めるための流れを整理します。

いきなり全文を書くのではなく、まずは過去問を読み、課題を分解し、答案骨子を作り、原稿用紙の中で文章に整えます。

## このページでできること

- 答案練習の始め方を確認する
- まず使うページを選ぶ
- 練習テーマと設問形式を選ぶ
- 1回分の練習手順を確認する

## 答案練習の流れ

過去問を読む  
→ 課題を分解する  
→ 答案骨子を作る  
→ 600字または1200字で書く  
→ 原稿用紙で確認する  
→ 改善点を次に反映する

## まず使う3つのページ

| 使うページ | 役割 | 次にすること |
|---|---|---|
| [過去問トレンドマップ](/guides/engineer/past-exam-trend-map) | 何が問われるかを知る | テーマと設問形式を選ぶ |
| [課題分解マトリクス](/guides/engineer/issue-decomposition-matrix) | 問題文から課題と解決策を整理する | 背景、課題、解決策、リスクを書き出す |
| [答案骨子ビルダー](/guides/engineer/answer-structure-builder) | 答案型に沿って骨子を作る | 必須Ⅰ、Ⅱ-1、Ⅱ-2、Ⅲの型に当てはめる |

答案の見え方を確認したい場合は、[模範答案例](/guides/engineer/model-answer-examples) で通常テキストと24字×25行の原稿用紙プレビューを見比べます。

## 練習テーマの選び方

最初は、頻出テーマから1つ選ぶと進めやすくなります。

| テーマ | 練習しやすい形式 | 入口 |
|---|---|---|
| 在庫削減、需給調整、計画連携 | Ⅱ-2 | [Ⅱ-2型答案練習](/guides/engineer/practice-ii-2) |
| 生産性向上、設備ロス改善、品質不良低減 | Ⅱ-2 | [Ⅱ-2型答案練習](/guides/engineer/practice-ii-2) |
| 物流2024年問題、DX、QMS再構築 | 必須Ⅰ、Ⅲ | [課題分解マトリクス](/guides/engineer/issue-decomposition-matrix) |
| 専門用語や手法の説明 | Ⅱ-1 | [キーワードマップ](/guides/engineer/keywords) |

## 1回分の練習手順

1. 過去問トレンドマップでテーマを1つ選ぶ
2. 設問形式別の問われ方で、答案に必要な要素を確認する
3. 課題分解マトリクスで、背景、課題、解決策、リスクを整理する
4. 答案骨子ビルダーで、設問形式に合う骨子へ変換する
5. 600字または1200字を目安に本文を書く
6. 原稿用紙プレビューで、文字量と改行の見え方を確認する
7. 模範答案例と比べて、足りない観点を次回に反映する

## 設問形式を選ぶ

| 形式 | 何を練習するか | 現在の入口 |
|---|---|---|
| 必須Ⅰ型 | 社会課題を多面的に整理し、解決策、リスク、倫理まで展開する | [課題分解マトリクス](/guides/engineer/issue-decomposition-matrix) |
| Ⅱ-1型 | 用語、手法、特徴、留意点を600字程度で説明する | [キーワードマップ](/guides/engineer/keywords) |
| Ⅱ-2型 | 業務の進め方、留意点、工夫点を1200字以内で整理する | [Ⅱ-2型答案練習](/guides/engineer/practice-ii-2) |
| Ⅲ型 | 専門分野の課題を抽出し、解決策とリスクを論じる | [課題分解マトリクス](/guides/engineer/issue-decomposition-matrix) |
