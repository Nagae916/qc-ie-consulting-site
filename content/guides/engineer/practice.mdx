# 技術士 経営工学 問題演習

## 問題演習の形式を選ぶ

技術士第二次試験は、必須Ⅰ、Ⅱ-1、Ⅱ-2、Ⅲで問われ方、文字数、答案構成が異なります。まずは練習したい設問形式を選び、必要なキーワードと答案骨子へ進みます。

過去問トレンドで学習テーマを選び、設問形式別の問われ方で答案要素を確認したら、このページから演習形式を選んでください。

## 必須Ⅰ型で練習する

社会課題を多面的に整理し、最重要課題、解決策、リスク、技術者倫理、社会の持続可能性まで展開する1600字答案を練習します。

文字数の目安：1600字程度

主な答案構成：

- 背景・社会課題
- 多面的な課題
- 最重要課題と理由
- 解決策
- リスクと対策
- 技術者倫理・社会の持続可能性

個別練習ページは今後追加予定です。現時点では、[答案骨子の作り方](/guides/engineer/answer-structure-guide) と [設問形式別の問われ方](/guides/engineer/past-exam-question-patterns) を使って練習します。

## Ⅱ-1型で練習する

専門用語、手法、特徴、留意点を、600字程度で簡潔に説明する練習をします。定義だけでなく、目的、構成要素、適用場面まで短く整理することが重要です。

文字数の目安：600字程度

主な答案構成：

- 定義・概要
- 目的・狙い
- 特徴・構成要素
- 適用上の留意点
- 経営工学上の意義

個別練習ページは今後追加予定です。用語の確認は [キーワードマップ](/guides/engineer/keywords) と [最重要キーワード100](/guides/engineer/keyword-priority-100) から進めます。

## Ⅱ-2型で練習する

業務の進め方、留意点、工夫点を1200字以内で整理する練習をします。生産管理、SCM、品質改善、在庫削減、リードタイム短縮などの実務テーマと相性が高い形式です。

文字数の目安：1200字以内

主な答案構成：

- 調査、検討事項
- 分析、方針設定
- 実施手順
- 留意点、工夫点
- 関係者調整（サプライチェーン全体）
- 効果確認

[Ⅱ-2型答案練習へ進む](/guides/engineer/practice-ii-2)

## Ⅲ型で練習する

専門分野の課題を多面的に抽出し、最重要課題、解決策、リスクと対策を1600字程度で論じる練習をします。必須Ⅰよりも、選択科目の専門性と実務性を強く意識します。

文字数の目安：1600字程度

主な答案構成：

- 背景
- 多面的な課題
- 最重要課題と理由
- 解決策
- リスクと対策
- 技術者倫理・社会の持続可能性

個別練習ページは今後追加予定です。まずは [過去問トレンド地図](/guides/engineer/past-exam-trend-map) でテーマを選び、[答案骨子の作り方](/guides/engineer/answer-structure-guide) で構成を確認します。
