# 供給制約・人手不足に使える経営工学キーワード

## 1. このテーマが技術士試験で重要になる理由

人手不足、設備能力不足、物流制約、需要変動は、経営工学部門の答案で扱いやすい社会課題です。単に人員を増やすのではなく、能力、在庫、工程、物流、情報共有を全体最適で設計する視点が求められます。

## 2. 社会背景・白書背景

少子高齢化、熟練者不足、物流制約、調達不安定化により、供給能力は固定的ではなくなっています。白書背景としては、人手不足、生産性向上、サプライチェーン強靭化、中小企業の省力化投資と接続できます。

## 3. このテーマで使える主要キーワード

| キーワード | 答案での使いどころ |
|---|---|
| [SCM](/guides/engineer/scm) | 調達、生産、物流、販売を全体最適で整合する |
| [需給調整](/guides/engineer/demand-supply-adjustment) | 需要予測、生産能力、在庫方針を調整する |
| [リードタイム短縮](/guides/engineer/lead-time-reduction) | 滞留、段取、情報遅れを削減する |
| [TOC理論](/guides/engineer/toc) | ボトルネックを特定し、スループットを高める |
| OEE | 設備稼働、性能、良品率を総合的に改善する |
| 多品種混流生産 | 需要多様化に対応しつつライン負荷を平準化する |
| 平準化 | 需要変動をならし、負荷と在庫を安定させる |
| [DX](/guides/engineer/dx) | 工程データ、需要データ、人員配置を見える化する |
| [レジリエンス](/guides/engineer/resilience) | 外乱時にも供給を継続・復旧できる体制を作る |

## 4. 答案での使いどころ

背景では、人手不足と需要変動により納期遵守や生産性が低下している状況を示します。課題では、工程能力の偏り、部門間情報共有不足、属人的な計画立案を挙げます。解決策では、SCM、TOC、DX、KPIを組み合わせて、供給能力を見える化し、全体最適で改善する流れにします。

## 5. 課題抽出に使える視点

- 生産能力と需要の不一致
- ボトルネック工程の未把握
- 販売、生産、物流の計画不整合
- 熟練者依存と技能伝承不足
- 在庫バッファと納期のトレードオフ

## 6. 解決策に使える視点

- S&OPにより販売計画、生産計画、在庫方針を整合する
- TOCで制約工程を特定し、段取短縮や要員配置を重点化する
- ERP、MES、WMSを連携し、実需、在庫、工程進捗を共有する
- OEE、納期遵守率、仕掛量など共通KPIで改善を管理する

## 7. リスクと対策に使える視点

DXや工程再設計を急ぐと、現場負荷、データ品質低下、部門間対立が起こります。段階導入、現場教育、KPIレビュー、責任分担の明確化で定着を図る必要があります。

## 8. 技術者倫理・社会の持続可能性への接続

供給制約への対応は、顧客への安定供給、従業員の過重労働防止、取引先との公正な役割分担に関わります。技術者は、効率性だけでなく安全、品質、働きやすさ、公益を考慮して改善策を設計します。
