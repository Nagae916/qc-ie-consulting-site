# 技術士 経営工学 最重要キーワード100

技術士第二次試験 経営工学部門では、キーワードを単独で暗記するだけでは不十分です。過去問で問われたテーマ、関連キーワード、白書背景、コンピテンシー、答案用途を結びつけて理解する必要があります。

このページでは、過去問傾向と答案作成への使いやすさを踏まえ、最優先で理解すべき100語を第1版として整理します。今後、公式過去問PDFを年次別・科目別に整理し、実際の出題テーマ・設問形式と対応づけて改訂していく前提です。

現在は、令和元年度〜令和7年度の経営工学部門126件の過去問メタデータに基づく[過去問トレンド分析](/guides/engineer/past-exam-trend-map)と接続して、S/A/Bランクを見直していく運用にしています。実データ上では、生産性向上、サービス品質、需給調整、物流効率化、DX、品質管理が上位テーマとして現れています。

頻出キーワードとしては、KPI管理、S&OP、サービス設計、IE、物流2024年問題、SCM、QMS、データドリブンなどが確認されています。ただし、メタデータ分類は今後のレビューで精緻化するため、未確認の語は「関連重要語」「答案補強語」として扱います。

![技術士 経営工学 最重要キーワード100の優先順位ピラミッド](/images/guides/engineer/keyword-priority-pyramid.svg)

## 1. この図の使い方

この図は、最重要キーワード100を優先度別に整理したものです。まずSランクを確実に理解し、Aランクで答案の幅を広げ、Bランクで具体性と専門性を補強します。

## 2. まず見るポイント3つ

- Sランク20語を最優先にする
- Aランクで答案の幅を広げる
- Bランクで具体性を補強する

## 3. この100語リストの使い方

- まずSランク20語を優先して理解する
- Sランクは、個別ページ化を最優先する
- Aランクは、テーマ別ページや答案用途別ページに反映しながら順次個別ページ化する
- Bランクは、関連キーワード・答案補強語として使う
- 過去問で問われた語は、定義だけでなく「課題・解決策・リスク・倫理」まで説明できる状態にする
- 関連語は、問われたテーマを多面的に論じるために使う
- 実データの集計結果は、公開ページでは[過去問トレンドマップ](/guides/engineer/past-exam-trend-map)に要約して反映する

## 4. 優先順位の考え方

### 分類A: 過去問で問われたことがある用語・論点

SCM、QMS、物流2024年問題、DX、BCP、KPI、品質管理、生産管理、需給調整、原価管理、IE、工程改善、作業改善、在庫管理、物流管理などは、経営工学部門の過去問テーマと接続しやすい重要論点です。

### 分類B: 過去問テーマを解くための関連ワード

TOC理論、S&OP、リードタイム短縮、レジリエンス、データドリブン、ERP、MES、WMS、TMS、モーダルシフト、デカップリングポイント、ブルウィップ効果、OEE、平準化、JITなどは、問われたテーマを答案化するための関連語です。

### 分類C: 問われがちな質問形式に使いやすい概念

課題抽出、解決策、リスクと対策、技術者倫理、社会の持続可能性、白書背景に接続しやすい語を優先します。単語を覚えるのではなく、答案のどこで使うかを決めるためのリストです。

## 9. 問われがちな質問形式との対応

| 質問形式 | 使えるキーワード例 | 答案での使い方 |
|---|---|---|
| 多面的な課題を3つ抽出せよ | SCM、DX、QMS再構築、物流2024年問題、レジリエンス | 供給、品質、データ、人材、物流など複数観点に分ける |
| 最重要課題を1つ選び、理由を述べよ | KPI、SCM、QMS、BCP | 顧客影響、事業継続、品質リスク、改善効果の大きさで理由付けする |
| 解決策を複数示せ | TOC、S&OP、DX、SMED、KPI | 課題ごとに施策を対応させ、実施手順と評価指標を示す |
| 解決策を実施しても生じうるリスクと対策を述べよ | レジリエンス、BCP、データガバナンス、[在庫最適化](/guides/engineer/inventory-optimization) | 効率化の副作用、データ品質、現場負荷、供給停止に備える |
| 技術者倫理の観点から留意点を述べよ | 技術者倫理、QMS再構築、品質不正防止、CSR | 公益、安全、説明責任、法令順守、データの適正利用を示す |
| 社会の持続可能性の観点から必要な取り組みを述べよ | 脱炭素・GX、モーダルシフト、SCM、人的資本経営 | 環境、労働、安定供給、地域社会への影響を扱う |
| 実務上の留意点・工夫点を述べよ | [標準化](/guides/engineer/standardization)、教育訓練、[変更管理](/guides/engineer/change-management)、[デザインレビュー](/guides/engineer/design-review)、KPIレビュー | 現場定着、関係者調整、段階導入、レビューを示す |
| KPIを設定して評価せよ | KPI、KGI、OEE、[在庫回転率](/guides/engineer/inventory-turnover)、品質KPI | 成果指標と先行指標を分け、改善効果を測定する |
