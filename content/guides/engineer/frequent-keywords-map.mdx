# 技術士第二次試験 頻出キーワードマップ

## 1. このページの目的

このページは、技術士第二次試験 経営工学部門で答案作成に使いやすいキーワードを体系的に整理するページです。

技術士二次試験では、単に用語を知っているだけでは不十分です。

重要なのは、設問の背景を読み取り、多面的な課題を抽出し、経営工学の専門技術を用いて解決策を示し、リスクと対策、技術者倫理、社会の持続可能性まで説明することです。

そのため、このページではキーワードを次の観点で整理します。

- どのテーマに関係するか
- どの課題に使えるか
- どの解決策に接続できるか
- どのコンピテンシーを示しやすいか
- 答案でどう表現できるか

| テーマ | 関連キーワード |
|---|---|
| QMS再構築・品質保証 | QMS、プロセスアプローチ、リスクに基づく考え方、CAPA、内部監査、品質不正防止 |
| 生産性向上 | IE、ECRS、標準作業、ラインバランシング、ボトルネック、OEE |
| DX・データ活用 | IoT、MES、BI、データ基盤、AI、予知保全、データドリブン経営 |

| サプライチェーン強靭化 | BCP、複数購買、在庫戦略、リードタイム短縮、リスク分散 |
| 人材育成・技能伝承 | OJT、Off-JT、多能工化、技能マップ、ナレッジマネジメント |
| カーボンニュートラル・省エネ | LCA、CO2排出量、エネルギー原単位、省エネ設備、グリーン調達 |
| リスクマネジメント | FMEA、FTA、リスクアセスメント、BCP、変更管理 |
| 標準化・業務プロセス改善 | SOP、標準化、見える化、業務フロー、BPR |
| コンプライアンス・倫理 | 品質不正、データ改ざん、説明責任、公益確保、持続可能性 |

## 3. 答案作成での使い方

頻出キーワードは、答案の中で以下のように使います。

| 答案の段階 | キーワードの使い方 |
|---|---|
| 背景整理 | 社会・産業・組織が抱える課題を説明する |
| 課題抽出 | 多面的な観点から技術課題を整理する |
| 解決策 | 経営工学の専門技術として提示する |
| リスク | 解決策を実行したときに生じる副作用を示す |
| 対策 | モニタリング、標準化、教育、レビューで補完する |
| 倫理・持続可能性 | 公益、安全、環境、人材、説明責任に接続する |

## 6. 600字答案に使うときの型

近年、製造業では人手不足、需要変動、品質要求の高度化により、従来の経験依存型の管理では安定した品質と生産性を維持しにくくなっている。このため、経営工学の観点から、工程データの見える化、標準作業の再設計、リスクに基づくQMS改善を組み合わせ、現場改善を継続的に進めることが重要である。施策実行時には、過度なシステム依存、現場負荷の増加、データ品質低下などのリスクが生じるため、教育訓練、運用レビュー、KPI管理により定着を図る。技術者は、公益確保と持続可能性を踏まえ、品質・コスト・納期・安全・環境を総合的に評価し、組織全体の改善能力を高める必要がある。
