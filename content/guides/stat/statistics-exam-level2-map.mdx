# 統計検定2級 学習マップ

## 1. このページの目的

このページは、統計検定2級の学習範囲を、出題単元・問われ方・学習優先度の観点から整理するページです。

統計検定2級では、単に公式を暗記するだけでは不十分です。グラフや表を読み取り、適切な統計量を選び、確率分布・推定・検定・回帰分析を使って判断する力が求められます。

このサイトでは、統計検定2級を次の観点で整理します。

- データを読む力
- ばらつきを扱う力
- 確率分布を使う力
- 推定・検定を判断する力
- 相関・回帰を解釈する力
- 問題文から必要な手法を選ぶ力

## 2. 統計検定2級の主要単元

| 大分類 | 主な単元 | 問われ方 |
|---|---|---|
| 記述統計 | 平均、中央値、分散、標準偏差、箱ひげ図、ヒストグラム | グラフ・表から統計量や分布の特徴を読む |
| 2変量データ | 散布図、相関係数、共分散、回帰直線 | 相関の向き・強さ、回帰係数の意味を読む |
| 確率 | 条件付き確率、独立、ベイズ、期待値、分散 | 文章から確率モデルを立てる |
| 確率分布 | 二項分布、正規分布、t分布、カイ二乗分布、F分布 | 分布の使い分けと確率計算 |
| 標本分布 | 標本平均、標本比率、中心極限定理 | 標本統計量のばらつきを理解する |
| 推定 | 信頼区間、母平均、母比率、母分散 | 推定式と区間の意味を判断する |
| 仮説検定 | z検定、t検定、比率の検定、片側/両側検定 | 帰無仮説、対立仮説、p値、有意水準を判断する |
| 分散分析 | ANOVA、F値、多重比較 | 複数群の平均差をばらつきの分解で見る |
| カイ二乗検定 | 適合度検定、独立性の検定 | 度数表から検定統計量を考える |
| 線形モデル | 単回帰、重回帰、決定係数、回帰係数 | 出力表を読み、係数・p値・決定係数を解釈する |
| 時系列 | 移動平均、自己相関、季節性 | 時系列グラフやコレログラムを読む |

## 4. 学習順序

1. 記述統計・可視化
2. 確率・期待値・分散
3. 確率分布
4. 標本分布
5. 推定
6. 仮説検定
7. 相関・回帰
8. 分散分析・カイ二乗検定
9. 時系列・線形モデル出力の読解

## 5. 参考書風の解説テンプレ

統計検定2級の問題解説は、以下のテンプレートで整理します。

1. 学習単元
2. 問題で問われていること
3. 解法の方針
4. 図解・式の意味
5. 選択肢判定
6. 正解
7. つまずきポイント
