# 単回帰分析を学ぶ

1つの説明変数が1つの目的変数に与える影響を分析する単回帰分析について、回帰直線のイメージと重要な概念、計算方法を整理します。

## 主要概念

### 回帰直線の傾きの求め方

最小二乗法による傾き $\hat{\beta}_1$ は **説明変数 $x$ と目的変数 $y$ の共分散** を **説明変数 $x$ の分散** で割ったものに相当します。

<div className="not-prose bg-gray-50 rounded-md p-3 overflow-x-auto">
<pre className="whitespace-nowrap text-sm md:text-base">
{`  \u005Chat{β}_1 = Σ(x_i- x̄)(y_i- ȳ)  /  Σ(x_i- x̄)^2  =  S_xy / S_xx`}
</pre>
</div>

- `S_xy` … $x$ と $y$ の偏差積和（共分散）
- `S_xx` … $x$ の偏差平方和（分散）

### 分散分析（ANOVA）の読み方

| 変動要因 | 偏差平方和 (SS) | 自由度 (df) | 分散 (MS)           | F比 |
|---|---:|---:|---:|---:|
| 回帰 (Regression) | `SS_R` | `1`   | `MS_R = SS_R / 1`   | `F = MS_R / MS_E` |
| 残差 (Error)      | `SS_E` | `n-2` | `MS_E = SS_E/(n-2)` |  |
| 全体 (Total)      | `SS_T = SS_R + SS_E` | `n-1` |  |  |

**F検定の帰無仮説**：$\beta_1 = 0$（回帰に意味なし）。棄却されればモデルは有意。

## 偏差平方和の分解と図解

全体平方和は **回帰による平方和** と **残差平方和** に分かれます。

<div className="not-prose bg-gray-50 rounded-md p-3 overflow-x-auto">
<pre className="whitespace-nowrap text-sm md:text-base">
{`  S_yy = Σ(y_i - ȳ)^2  =  Σ(ŷ_i - ȳ)^2  +  Σ(y_i - ŷ_i)^2
           ^^^^^^^^^^^     ^^^^^^^^^^^
             回帰による平方和        残差平方和`}
</pre>
</div>
