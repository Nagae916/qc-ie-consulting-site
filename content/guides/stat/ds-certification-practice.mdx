# データサイエンティスト検定 4択演習

## 1. この演習の使い方

このページは、データサイエンティスト検定対策用の4択演習です。

公式範囲や過去問系資料で問われやすい観点を参考にしつつ、問題文・選択肢・解説はn-ie-qclab向けのオリジナル類題として作成しています。

## 2. 演習の目的

用語暗記だけでなく、適切な手法選択、ケース判断、データ活用の考え方を鍛えることを目的にします。

間違えたテーマはカテゴリと skill を確認し、関連ガイドで復習してください。

## 4. 復習のポイント

- 間違えた問題は category と skill を確認する
- relatedGuide がある場合は該当ガイドに戻る
- 正解した問題でも、説明できないものは復習対象にする
- 他の選択肢がなぜ不適切かを自分の言葉で説明する

## 5. 今後の問題追加方針

- 土曜ごとに問題を追加していく
- 過去問・公式テキスト・公式模擬問題がある場合は、その出題傾向に沿って問題を追加する
- ただし問題文や選択肢はオリジナル化する
- 問題数の目標はまず30問、次に50問、最終的に100問以上
