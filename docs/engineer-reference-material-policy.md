# 技術士 経営工学部門 優先参照資料方針

## 1. 基本方針

アップロード済みの「技術士第二次試験対策講座（経営工学部門）」を、経営工学部門の傾向分析、例題生成、答案骨子作成における優先参照資料として扱う。

また、「第二回解答のポイント」「第二回添削課題」も、答案構成、設問解釈、添削観点、再発防止ルールの基準資料として扱う。

過去問メタデータ、傾向分析、例題生成、答案骨子出力を行う際は、単にタグや問題パターンを機械的に作るのではなく、講座テキスト、解答ポイント、添削フィードバックと整合しているかを確認する。

## 2. 確認する観点

以下を確認する。

1. 技術士第二次試験対策講座の内容と矛盾していないか
2. 必須科目Ⅰ、選択科目Ⅱ-1、Ⅱ-2、Ⅲの答案型と整合しているか
3. 経営工学部門らしい観点、用語、手法が含まれているか
4. 解答例・解答のポイントと比べて、構成が浅くなっていないか
5. 添削課題で指摘された弱点を再発しない設計になっているか
6. 白書・法令・政策情報を使う場合は、一次情報と整合しているか

## 3. 過去問メタデータへの適用

過去問メタデータを登録・修正する際は、以下を確認する。

- questionPattern が講座で示される設問解釈とずれていないか
- requiredActions が答案で実際に求められる動作を表しているか
- themeTags / methodTags / policyTags / lawTags が経営工学部門の答案で使える語になっているか
- summary が単なる問題概要ではなく、答案構成につながる要約になっているか
- skeletonTemplateId が答案型と一致しているか

## 4. 傾向分析への適用

PastExamTrendMap で傾向分析を表示する際は、件数だけで学習優先度を決めない。

頻出タグや設問パターンに加えて、講座テキストで重視される答案構成、添削で指摘されやすい弱点、経営工学部門らしい観点を踏まえて、学習優先度や推奨アクションを整理する。

## 5. 例題生成への適用

例題生成では、公式過去問の問題文を転載しない。

ただし、例題の妥当性確認には、講座テキスト、解答のポイント、添削課題を優先的に参照する。

例題は以下を満たすようにする。

- 必須科目Ⅰ、選択科目Ⅱ-1、Ⅱ-2、Ⅲの答案型に合っている
- 問題文の要求動作が明確である
- 経営工学部門らしい対象、観点、手法が含まれている
- 課題抽出、最重要課題、解決策、リスク、倫理・持続可能性のつながりが崩れていない
- 白書・法令・政策タグを使う場合は、一次情報と矛盾しない

## 6. AnswerStructureBuilder への適用

AnswerStructureBuilder は、一般的な作文テンプレートではなく、講座テキスト、解答のポイント、添削課題で整理された答案作成ルールに従って設計する。

特に以下を重視する。

- 問題点、技術課題、解決策を混同しない
- 課題は抽象語で終わらせない
- 最重要課題は抽出した課題の中から選ぶ
- 解決策は最重要課題に直接対応させる
- リスクは施策実施後に新たに生じる副作用として書く
- 技術者倫理・社会の持続可能性は、テーマ固有の具体語で書く
- 添削課題で指摘された弱点を再発しないようにする

## 7. 著作権・参照上の注意

過去問本文、講座テキスト、解答ポイント、添削課題の全文転載は行わない。

サイト上では、学習用に再構成した要約、分類、確認観点、オリジナル例題、答案骨子作成支援として扱う。

公式過去問を確認する場合は、日本技術士会の公式PDFリンクを参照する。
