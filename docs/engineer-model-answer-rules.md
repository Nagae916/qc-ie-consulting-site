# 技術士 経営工学部門 模範答案・ブラッシュアップ答案ルール

## 目的

技術士第二次試験 経営工学部門の模範答案、ブラッシュアップ答案、別解答案は、通常テキスト版と原稿用紙プレビュー版の両方で使える形に整理する。

答案例は、問題文、模範解答、ユーザー答案、添削コメントの丸写しではなく、学習用に再構成したオリジナル答案例として扱う。

## 原稿用紙の基準

- 技術士二次試験の答案用紙は、24字×25行＝600字を1枚の基準とする。
- 1枚答案は、改行や見出しによる空白マスを考慮し、本文550〜575字程度を推奨目安とする。
- 2枚答案は、1200字ぴったりではなく、本文1100〜1150字程度を推奨目安とする。
- 3枚答案は、本文1650〜1725字程度を推奨目安とする。
- 原稿用紙プレビューでは、改行によって発生する空白マスも含めて、答案用紙に収まるかを確認する。

## 答案例データに持たせる情報

答案例には、本文だけでなく次の情報を持たせる。

- 出典名
- 試験区分
- 分野
- テーマ
- 設問パターン
- 答案フレームID
- 想定原稿用紙枚数
- 目標本文文字数
- 評価観点
- 使用キーワード
- 改善意図
- 原稿用紙上の注意点

## 答案例作成ルール

- Ⅱ-1型は、600字1枚を想定し、定義、構成、特徴、留意点、比較要求への回答を簡潔にまとめる。
- 比較説明型では、比較対象、共通点、相違点、メリット、デメリット、適用場面、有効な段階を明確にする。
- Ⅱ-2型は、1200字2枚を想定し、調査・検討事項、進め方、留意点・工夫点、関係者調整、KPI・効果確認を分ける。
- Ⅱ-2型では、手法名だけを列挙せず、目的、対象、効果確認と結びつける。
- 関係者調整では、誰と、何を、どの情報に基づいて調整するかを具体化する。
- 欠品、品質悪化、現場負荷、部門最適など、改善活動の副作用にも触れる。

## 将来の利用方針

作成した答案例データは、将来的に以下へ接続できるようにする。

- AnswerStructureBuilder
- 原稿用紙プレビュー
- Ⅱ-1型答案練習
- Ⅱ-2型答案練習
- 添削課題の復習ページ
- 模範答案・別解答案の比較表示
