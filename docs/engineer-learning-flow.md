# 技術士 経営工学部門 学習フロー・ページ役割

## 1. 技術士領域の目的

技術士 経営工学部門のページ群は、過去問傾向を把握し、重要キーワードを理解し、設問形式に応じて答案骨子を作り、実際の答案練習まで進めることを目的とする。

## 2. 技術士学習フロー

1. 使い方を知る
2. 過去問傾向を見る
3. 設問形式を理解する
4. 重要キーワードを学ぶ
5. 答案骨子を作る
6. 問題演習をする
7. 答案を改善する

## 4. トレンドマップの役割

トレンドマップの役割：
過去問126件から、優先して学ぶテーマ・キーワード・分野を把握する。

トレンドマップでやること：

- 頻出テーマを見る
- よく使うキーワードを見る
- 必要な学習分野を確認する
- 次に学ぶページへ進む

トレンドマップでやらないこと：

- 問題演習
- 問題生成
- 詳細フィルタ
- 答案作成
- 長い入力欄の表示

## 5. 問題演習ハブの役割

問題演習ハブ `/guides/engineer/practice` は、技術士 経営工学部門の演習入口である。
ここでは、テーマ・設問形式・キーワードのいずれかから演習へ進めるようにする。

将来的には、以下の機能を追加できる。

- テーマ別答案練習
- 設問形式別答案練習
- キーワード指定の骨子作成
- 600字答案練習
- 1600字答案練習
- 添削観点チェック

## 6. 再発防止ルール

- マップページに問題生成UIを置かない
- 入口ページに詳細分析表を置かない
- 個別解説ページに大量の全体導線を置かない
- 演習ページにサイト全体説明を置きすぎない
- 各ページの冒頭に「このページの役割」を置く
- 各ページ末尾に「次に読むページ」を置く
- 新規ページを作る前にページ役割テンプレートを確認する
