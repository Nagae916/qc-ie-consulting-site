# 第3回添削課題から抽出する答案改善ルール

## 1. この資料を参照する目的

第3回添削課題は、技術士第二次試験 経営工学部門の選択科目Ⅱ-1、Ⅱ-2で、設問要求に沿って答案を組み立てるための学習用メタデータとして参照する。

本文、模範解答、添削コメントは転載しない。サイト内では、問題テーマ、設問パターン、答案フレーム、評価観点、改善点、推奨学習内容として整理し、AnswerStructureBuilder や答案練習ページで再利用できる形にする。

## 2. 選択科目Ⅱ-1型で改善すべきポイント

選択科目Ⅱ-1では、用語や手法の定義だけでなく、構成、メリット、デメリット、適用場面を短く整理する必要がある。比較説明型では、共通点と相違点を明確にし、どの段階で効果を発揮するかまで述べる。

第3回添削課題の「ガントチャートとアローダイアグラム」では、以下が重要である。

- ガントチャートとアローダイアグラムの構成を分けて説明する
- メリット・デメリットをそれぞれ整理する
- 両者の違いを、表現方法、依存関係の把握、進捗管理のしやすさで比較する
- 計画・分析段階と実行・進捗管理段階のどちらで有効かを示す
- 図を使う場合も、図が何を意味するかを文章で補足する

設問要求は、単なる用語説明ではなく「比較説明型」として扱う。答案では、ガントチャートを作業項目と時間軸で進捗を見える化する手法、アローダイアグラムを作業順序、先行後続関係、クリティカルパスを分析する手法として整理する。

答案ブロックは以下を標準とする。

- 定義
- 構成
- 目的
- メリット
- デメリット
- 共通点
- 相違点
- 適用場面
- 有効な段階
- 経営工学上の活用場面

避けるべき答案は、定義だけで終わる答案、メリットとデメリットが混在する答案、比較対象の違いが曖昧な答案、問題文で問われた段階に答えていない答案、図だけで説明が不足する答案である。

## 3. 選択科目Ⅱ-2型で改善すべきポイント

選択科目Ⅱ-2では、実務で業務を進める前提で、調査・検討事項、実施手順、管理項目、留意点、関係者調整、効果確認を整理する必要がある。

第3回添削課題の「見込み生産型企業の在庫削減プロジェクト」では、以下が重要である。

- 調査・検討事項と実施手順を混ぜずに整理する
- 需要予測、販売計画、生産計画、在庫計画を連動させる
- 在庫削減だけでなく、欠品リスク、納期、顧客サービス水準を考慮する
- 営業、生産、購買、倉庫、経営層など、関係者ごとに調整内容を明確にする
- 在庫回転率、欠品率、計画達成率、リードタイムなどのKPIで効果確認する
- 手法名を挙げるだけでなく、目的、対象、期待効果を説明する

設問要求は、在庫削減プロジェクトの進め方を、設問(1)(2)(3)に対応して分けて述べる形式である。答案では、調査・検討事項、実施手順、関係者調整を混ぜずに見出しで区切る。

標準構成は以下とする。

### (1) 調査・検討事項

- 現状把握
- データ収集
- 問題の要因分析
- 検討すべき指標
- 対象範囲の特定

### (2) 進め方・留意点

- 改善方針の設定
- 実施手順
- 使用する経営工学手法
- 管理指標・KPI
- 留意点・工夫点
- 効果確認

### (3) 関係者との調整方策

- 関係者の列挙
- 役割分担
- 共有すべき情報
- 調整会議・レビューの仕組み
- 合意形成
- 継続的な改善サイクル

避けるべき答案は、調査事項と実施手順が混在する答案、手法名だけを列挙する答案、関係者が抽象的な答案、KPIや効果確認がない答案、欠品リスクや副作用に触れていない答案である。

## 4. ユーザー答案から読み取れる強み

今回の要約から読み取れる強みは以下である。

- ガントチャート、アローダイアグラム、需要予測、安全在庫、ABC分析、ERP連携など、経営工学の用語を答案に使おうとしている
- 図や工程管理の観点を使って説明しようとしている
- 在庫削減を単独施策ではなく、販売、生産、購買、倉庫を含む計画連携の問題として捉えている
- 関係者調整の必要性を意識できている

## 5. ユーザー答案から読み取れる改善点

改善点は、知識不足というよりも、設問要求に合わせた答案構成の明確化である。

- 比較説明型では、定義、共通点、相違点、適用場面を分ける
- メリット・デメリットを混在させず、対象ごとに整理する
- 調査・検討事項と実施手順を分ける
- 関係者調整では、誰と何を調整するかを書く
- 手法名だけでなく、目的、対象、効果を述べる
- KPIで効果確認まで示す

## 6. 今後の推奨学習内容

選択科目Ⅱ-1向けには、工程管理とプロジェクト管理の基礎を補強する。

- ガントチャート
- アローダイアグラム
- PERT
- 工程管理
- 日程計画
- 進捗管理

選択科目Ⅱ-2向けには、見込み生産、在庫削減、需給調整、部門間調整を補強する。

- 在庫回転率
- 安全在庫
- 発注点
- 需要予測
- 販売計画
- 生産計画
- MRP
- ERP
- S&OP
- ABC分析
- 欠品リスク
- 過剰在庫
- リードタイム
- 部門間調整

## 7. AnswerStructureBuilder に反映すべきルール

大規模なUI変更は不要だが、将来的には以下を反映する。

- 選択科目Ⅱ-1型では、比較対象、共通点、相違点、適用場面、有効な段階を入力できるようにする
- 選択科目Ⅱ-2型では、調査・検討事項、実施手順、関係者、調整方策、KPIを分けて入力できるようにする
- correction-examples.json の builderHints を、答案フレームごとの補助ヒントとして表示できるようにする
- Ⅱ-1の比較説明型では、共通点、相違点、適用場面、有効な段階を入力時に確認できるようにする
- Ⅱ-2の手順説明・留意点型では、誰と、何を、どの頻度で、どの情報を共有するかまで具体化を促す

## 8. answer-frame-rules.json に反映すべきルール

選択科目Ⅱ-1型には、比較説明型に対応するため、比較軸、メリット・デメリット、適用場面、有効な段階を評価観点として追加する。

選択科目Ⅱ-2型には、調査・検討事項と実施手順の分離、関係者調整、KPIによる効果確認、在庫削減と欠品リスクの両立を評価観点として追加する。

今回のメタデータは、`public/data/engineer/correction-examples.json` に蓄積し、今後の添削課題、模範解答例、答案骨子ヒントのバックデータとして使う。公開ページでは、PDF本文、手書き答案、模範解答、添削コメントをそのまま掲載せず、学習用の評価観点と改善ルールとして再構成する。
